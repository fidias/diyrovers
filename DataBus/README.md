mbed lpc1768 pinouts
====================

UART
----

 * UART0 is the USB connection
 * UART1 
    * rx = p14 
    * tx = p13
 * UART2 
    * rx = p27
    * tx = p28
 * UART3
    * rx = p18
    * tx = p17

I2C
---
 * sda = p9 
 * scl = p10

Encoders
--------
 * A (Outermost)
    * LEFT = p30 
    * RIGHT = p29
 * B (Innermost) unavilable


GPIO
----
 * Digital
    * D0 = p11 
    * D1 = p12
    * D2-D9 = unavailable
 * PWM 
    * P1 = p26 
    * P2 = p25
 * Analog
    * A0 = p15
    * A1 = p16
    * A2 = p19
    * A3 = p20
